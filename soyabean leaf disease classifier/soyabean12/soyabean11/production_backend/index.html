About This System

This AI-powered soybean disease detection system combines cutting-edge computer vision technology with deep learning to provide accurate, fast, and reliable disease diagnosis for agricultural applications.

The system uses a two-stage pipeline: YOLOv8 for disease region detection and a V2 Ensemble (EfficientNet-B4 + ResNet152) for precise classification across 17 disease types including bacterial blight, rust, mosaic virus, and more.
